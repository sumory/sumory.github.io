我的开源代码

  • Lor Framework : 一个运行在OpenResty上的基于Lua编写的Web框架

  • Gru : 即时通讯集群解决方案

  • confd : 统一配置管理工具

  • Moklr : 一个HTTP API测试和监控工具

  • mkd : markdown cli工具，同时提供类似于lincell 的编辑功能

  • lincell : 一行命令创建远程文件编辑器

  • beauty : Console、字符串美化模块

  • xgen : 使用模板生成代码、项目脚手架，可自行定制模板.

  • gb : Github命令行操作工具.

  • baseN : Node.js进制转换、编解码模块

  • nodis : 基于Redis和Node.js的前缀搜索、模糊搜索服务

  • BTC : 比特币矿池及配套系统

  • LTCBlockchain : 查询莱特币区块、交易、地址等信息的系统

  • UC : 分布式系统中ID生成，唯一性字段值管理通用模块

  • sumorio : 基于Node.js的SNS

  • social_auth : 新浪微博、人人网、QQ等第三方Oauth授权
