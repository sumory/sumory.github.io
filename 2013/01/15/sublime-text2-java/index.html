在Sublime Text2中开发Java

  • editor
  • java

  • 第一步

    确保Java Path已设置

  • 第二步

    编写脚本runJava.bat，放在jdk的bin目录下

  • 第三步

    • Preferences > Browse Packages.. 打开sublime的包目录
    • 打开Java文件夹
    • 打开JavaC.sublime-build，并替换"cmd": ["javac", "$file"]为"cmd": ["runJava.bat", "$file"]

  • 第四步

    编写测试代码，Ctrl+B编译运行
